# KeyFarm - 鍵盤農場設計文件

## 概念

螢幕上常駐一個 HHKB 鍵盤形狀的小農場，透明背景置頂顯示。用戶每次敲鍵盤，對應位置的耕地就會成長。桌面寵物/裝飾用途，療癒風格。

## 視覺風格

可愛卡通風格，圓潤的果樹和水果。

## 核心機制

### 耕地生命週期

每格對應一個按鍵：

```
空地 →[敲擊 5 次]→ 灑水中 →[敲擊 15 次]→ 發芽 →[敲擊 30 次]→ 果樹 →[敲擊 50 次]→ 結果
  ↑                                                                              │
  └──────────────────── 滑鼠經過採收 ←──────────────────────────────────────────────┘
```

### 播種

結果的水果種類隨機（蘋果、橘子、櫻桃、葡萄等）。

### 採收

滑鼠移到結果的格子上觸發採收動畫，然後回到空地。

## HHKB 配列

POC 先處理可顯示字元鍵（字母、數字、符號）約 47 格。修飾鍵和 Space 先不做耕地。

## 技術架構

- **Tauri v2** — 桌面 app 框架
- **Rust 後端** — `rdev` crate 全域鍵盤監聽
- **前端** — React + Canvas
- **存檔** — 本地 JSON

### 視窗行為

- 透明背景 + 置頂 + 可拖曳
- System tray icon（顯示/隱藏、退出）
- 快捷鍵切換顯示/隱藏
